This project is a simple and responsive image card layout built using HTML and CSS.

Features
1.Clean and minimal UI
2.Responsive layout using `vh` and `vw`
3.Gradient button design
4.Smooth hover and active effects
5.No Flexbox or Grid used (pure CSS)

Technologies Used
1.HTML5
2.CSS3

HTML Code Overview
1.Uses semantic HTML structure
2.Image and button wrapped inside a container

CSS Styling Highlights 1.`vh` and `vw` units for responsive spacing
2.Linear gradient background for the button
3.Smooth hover transition with shadow and lift effect
4.Box shadow for card depth

On hovering the button:
1.Background color changes
2.Button slightly lifts up
3.Shadow becomes more prominent

Responsive Behavior
1.Layout adapts to different screen sizes
2.Button height scales using viewport units
3.Image remains fully responsive

How to Use
1.Clone the repository
2.Open `index.html` in your browser
3.Customize colors, image, or button text as needed

This project is open-source and free to use.
